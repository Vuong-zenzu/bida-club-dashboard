# Bida Club Dashboard

## Overview
Dashboard quản lý Bida Club, bao gồm các chức năng tính tiền giờ, menu, báo cáo doanh thu.

## Features
- **Quản lý bàn:** Bắt đầu/Kết thúc/Tạm dừng/Chuyển bàn.
- **Tính tiền:** Tự động tính tiền giờ + Order menu.
- **Thanh toán:** Chi tiết hóa đơn, hỗ trợ nhiều phương thức (Tiền mặt, QR, Chuyển khoản).
- **Settings:** Cấu hình giá giờ, menu, giao diện (Dark/Light).
- **Report:** Báo cáo doanh thu ngày/tuần/tháng (Lưu trữ LocalStorage).

## Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
- Recharts (Charts)

## Setup
```bash
npm install
npm run dev
```
